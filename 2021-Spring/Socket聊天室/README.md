### 项目介绍

桌面客户端采用了 Qt 这个 C++ 框架来编写，服务端采用了 Go 来编写，完成了一个比较简单的聊天室。

## 该聊天室有以下几个特点:
- [x] 能够保存最近 3 天的消息记录    
- [x] 消息到达客户端后有消息提醒    


客户端跨平台，目前支持了 Windows 、 Linux 、Android 平台。

### 如何体验

去客户端仓库中下载对应系统的客户端，然后在服务端信息一栏输入`qaq.iinfo.dev:8080`或主机名`qaq.iinfo.dev`+端口`8080`，之后在用户名框中输入你喜欢的用户名（暂不支持用户名重复检测），再连接即可。
